Multiple lines

When using .bfg--wrap modifier, .bfg__box elements can be displayed on multiple lines.
In this case, the boxes may leave outside the .bfg block, and scrollbar may appear at the parent node of the grid.

Note for Chrome:
When using .bfg--col modifier, you have to explicitly set the width of the .bfg__box elements, otherwise content like text will increase indefinitely in width.
